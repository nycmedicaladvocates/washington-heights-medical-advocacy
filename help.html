Request help (form)

Step 1: Create a Google Form. Step 2: paste the embed link below (instructions under this box).

How to create the Google Form + embed it

  1. Go to forms.google.com → Blank form
  2. Title: “Washington Heights Medical Advocacy – Request Help”
  3. Add questions:
    • Language / Idioma (English / Español)
    • Name (optional)
    • Phone or Email
    • What do you need help with? (appointment / NYC Care / clinic choice / directions / other)
    • Best time to reach you
  4. Click Send → choose the < > embed icon → copy the embed link
  5. Paste that link into the iframe src above (replace the placeholder text)
